Mysterious Restaurant: Match-3

📱 Mobile 🚀 Released
🎮 Game Page

About the Game

Inherit a timeless restaurant, and connect with your customers through unique challenge. A restaurant simulation game with match-3 and merge mechanics.

My Role

  • Game Designer
  • Unity Developer

Responsibilities

  • Designed the game and onboarding
  • Implemented game systems and UI in Unity
  • Maintained documentation
  • Playtested and iterated based on feedback
  • Tracked analytics data

Technologies Used

  • Unity Engine
  • C#
  • Figma (for UI/UX)
  • Firebase

Development Journey

This project originally began as a trivia-based restaurant game. However, we later pivoted to a match-3 gameplay style to better support the restaurant theme and improve engagement.

  • Integrating trivia into a restaurant setting was difficult. The pivot to match-3 and merge mechanics allowed for a more natural and engaging gameplay loop.
  • Learned the importance of adapting design based on player feedback and analytics, and how to blend different genres for a unique experience.
  • Developed a day/night cycle system and dynamic customer behaviors to keep gameplay fresh and replayable.
